# Images Directory

This directory contains images related to the AI Document Check project, including:
- Damaged shipping container examples
- Non-conformity documentation photos
- Reference images for AI training/validation

## Image Files

Please add your image files to this directory. Based on the descriptions provided, the following images are expected:

1. **damaged-container-msc.jpg** - Yellow MSC container with significant dents and rust
2. **damaged-container-holes.jpg** - Light-colored container with holes and tears
3. **stacked-containers-rust.jpg** - Stack of weathered reddish-brown containers
4. **container-yard-muddy.jpg** - Dark grey container with "GOOSE NECK" label on muddy ground
5. **container-stack-colored.jpg** - Multi-colored container stack with Terminal Tracker branding

## Usage

These images can be used for:
- AI model training and validation
- Documentation and examples
- Testing damage detection algorithms
- Reference material for non-conformity reporting

## File Naming Convention

Suggested naming: `{type}-{description}-{number}.{ext}`
- Example: `container-damaged-msc-01.jpg`
- Example: `container-rust-stack-01.jpg`
